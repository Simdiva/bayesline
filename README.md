bayesline
=========

A Multinomial Bayesian Classification for Language Identification.

`bayseline.py` is used as a baseline system for the Discrimination of Similar Languages (DSL) shared task in COLING-2014 ((http://corporavm.uni-koeln.de/vardial/sharedtask.html).
